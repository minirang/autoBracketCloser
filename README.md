# autoBracketCloser

A tiny JavaScript utility that automatically closes brackets and quotes
in text inputs and textareas.

---
##  Features

- Automatically closes brackets: `()`, `{}`, `[]`
- Automatically closes quotes: `'`, `"`
- Deletes empty pairs together with one Backspace
- Works with `<input>` and `<textarea>`
- Light and easy to use

---
##  Installation

This library is not published to npm.

Just copy the code and include it in your project:

```html
<script src="autoBracketClose.js"></script>
```
